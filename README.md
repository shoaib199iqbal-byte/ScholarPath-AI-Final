ScholarPath AI

ScholarPath AI is an AI-powered student guidance application designed to help students explore suitable academic and career pathways based on their interests, education, skills, and goals.

🚀 Live Demo

Open ScholarPath AI https://scholar-path-ai-final.vercel.app

📌 About the Project

Choosing the right academic and career path can be challenging for students. ScholarPath AI provides personalized guidance by using artificial intelligence to analyze student information and suggest relevant academic and career pathways.

The application is designed to make career and academic exploration easier, more accessible, and more personalized.

✨ Features

* 🤖 AI-powered student guidance
* 🎓 Personalized academic pathway suggestions
* 💼 Career pathway recommendations
* 📊 Student profile and information analysis
* 🔍 AI-based advisor matching
* 📱 User-friendly and responsive interface
* 🌐 Live web application deployment

🧠 AI Feature

ScholarPath AI uses Google’s Gemini AI to provide intelligent and personalized recommendations.

The AI feature analyzes information provided by the user and generates relevant guidance based on their academic background, interests, skills, and career goals.

⚙️ Setup and Installation

1. Clone the Repository

git clone YOUR-GITHUB-REPOSITORY-LINK

2. Open the Project

cd ScholarPath-AI-Final

3. Install Dependencies

npm install

4. Configure Environment Variables

Create a .env.local file and add your Gemini API key:

GEMINI_API_KEY=your_api_key_here

Do not share or upload your API key publicly.

5. Run the Application

npm run dev

Open the local application in your browser.

🌐 Deployment

The application is deployed using Vercel and is available through the live demo link provided above.
